# Biological Interpretation

The recommended first biological result to inspect is the final evidence bundle:

```text
results/tables/10_biological_integration/final_evidence_bundle/global/final_biological_evidence_bundle.xlsx
```

The bundle brings together manuscript program summaries, the evidence priority matrix, cross-compartment atlas rows, WGCNA key module/supermodule context, microglia ROI targeted-signature drivers, QC flags, and the biological claims table. It is a final review layer only; it does not recompute upstream statistics, WGCNA construction, model formulas, or thresholds.

The biological claims table is an evidence index for manuscript drafting, not an automated conclusion engine. It is generated by `09_export_pride_journal/07_make_biological_claims_table.R` and validated against `inst/schemas/biological_claims_table.yml`.

Evidence grades:

| grade | meaning |
|---|---|
| A | Differential abundance plus enrichment plus module/network support. |
| B | Differential abundance plus enrichment support. |
| C | Enrichment-only or module-only support. |
| D | Exploratory evidence. |
| X | Confounded or unsafe for manuscript claim without additional review. |

Required guardrail columns:

- `claim_grade`
- `primary_evidence`
- `orthogonal_support`
- `major_limitation`
- `safe_interpretation`
- `unsafe_overinterpretation`

Allowed language should describe associations and convergent evidence. Avoid causal, mechanistic, or cell-intrinsic claims unless independent validation supports them.

Final-facing terminology:

- `neuron_neuropil`: region/layer-resolved neuron neuropil
- `neuron_soma`: region-resolved neuronal soma-enriched
- `microglia`: region-resolved microglia-enriched ROI/local microenvironment, not purified microglia

Use `microglia/immune` language only when the source rows show explicit immune-state evidence such as complement, phagolysosomal/lysosomal, inflammatory, MHC/interferon, cytokine/chemokine, or DAM-like terms. Targeted microglia signature overlap by itself should be described as ROI/local microenvironment evidence unless the driver columns support stronger wording.
